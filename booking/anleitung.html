Login & Zugang

Anmeldung: Klicken Sie oben rechts auf den Button "Konto", um das Kundenportal zu öffnen. Geben Sie Ihre registrierte E-Mail-Adresse und Ihr Passwort ein.

Registrierung: Falls Sie noch kein Konto haben, können Sie sich im Reiter "Neu registrieren" im selben Fenster anmelden.

Passwort vergessen: Nutzen Sie den Link im Login-Fenster. Sie erhalten eine E-Mail zum Zurücksetzen.

Sitzung: Nach dem Login bleiben Sie auf Ihrem Gerät für 30 Tage angemeldet.

Neue Buchung erstellen

Der Buchungsprozess ist in 4 einfache Schritte unterteilt:

  • Schritt 1 (Standort): Geben Sie die Adresse der Immobilie sowie Details wie Wohnfläche (m²) und Etagen/Ebenen ein. Diese sind wichtig für die Preisberechnung.
  • Schritt 2 (Dienstleistungen): Wählen Sie das gewünschte Paket (z.B. Basic, Premium, The Full View) und fügen Sie nach Bedarf Add-ons wie Drohnenaufnahmen oder Grundrisse hinzu.
  • Schritt 3 (Fotograf & Termin): Wählen Sie einen bevorzugten Fotografen oder lassen Sie das System den bestmöglichen zuweisen. Anschliessend können Sie im Kalender einen passenden, verfügbaren Termin auswählen.
  • Schritt 4 (Rechnung): Überprüfen Sie Ihre Rechnungsadresse (diese wird aus Ihrem Profil vorausgefüllt) und den Kontakt vor Ort für den Fotografen. Bestätigen Sie abschliessend die AGB.
Meine Aufträge verwalten

Im Kundenkonto finden Sie eine Liste all Ihrer vergangenen und zukünftigen Aufträge.

Detailansicht: Klicken Sie auf einen Auftrag, um diesen aufzuklappen.

Status-Bedeutungen:

  • Ausstehend: Buchung eingegangen, wartet auf Bestätigung.
  • Bestätigt: Termin fixiert, Fotograf ist zugewiesen.
  • Pausiert: Auftrag temporär gestoppt (z.B. wegen schlechtem Wetter). Wir melden uns für einen neuen Termin.
  • Erledigt: Shooting abgeschlossen, Daten in Postproduktion.
  • Abgeschlossen: Projekt beendet, finale Daten übergeben.
Termine verschieben oder stornieren

Stornieren: Öffnen Sie den Auftrag und klicken Sie auf 'Stornieren'. Bitte beachten Sie mögliche Gebühren gemäss AGB.

Verschieben: Nutzen Sie die Nachrichtenfunktion im Auftrag oder rufen Sie uns direkt an.

Kommunikation & Daten hochladen

Nachrichten senden: Bei jedem Auftrag finden Sie ein Feld 'Nachricht senden' für wichtige Infos an den Fotografen (z.B. 'Schlüssel unter der Fussmatte').

Daten hochladen: Oben im Kundenkonto finden Sie den Button 'Daten hochladen'. Nutzen Sie diesen, um Referenzbilder oder alte Grundrisse sicher an uns zu übermitteln.

Systemzugang & Login

Login: Öffnen Sie admin-booking.propus.ch. Nutzen Sie Ihre Mitarbeiter-Zugangsdaten.

Das System erkennt Ihre Rolle automatisch und blendet Admin-Funktionen aus.

Auftragsübersicht & Kalender

Auftragsliste: Eine tabellarische Übersicht aller Ihnen zugewiesenen Aufträge. Sie können diese nach Datum oder Status sortieren.

Kalender: Übersicht über geplante Shootings. Bei jeder neuen Buchung erhalten Sie eine E-Mail mit einer ICS-Datei. Diese enthält auch einen Direktlink, um den Auftrag im System sofort zu öffnen.

Auftrag abwickeln & Status

Klicken Sie auf das 'Auge'-Symbol, um Details wie Adresse, Paket und Kontakt vor Ort einzusehen.

  • Setzen Sie den Status auf 'Bestätigt', wenn Sie den Termin geprüft haben.
  • Setzen Sie den Status auf 'Pausiert', falls das Shooting nicht stattfinden kann (z.B. Wetter).
  • Setzen Sie den Status auf 'Erledigt' nach erfolgreichem Shooting.
Material Upload (Rohdaten)

Der Upload erfolgt direkt im Buchungssystem:

  1. Klicken Sie in der Liste beim Auftrag auf das Upload-Symbol.
  2. Wählen Sie den Ziel-Ordner (z.B. 'Unbearbeitete / Bilder').
  3. Ziehen Sie Ihre Dateien per Drag & Drop in das Feld. Erlaubt sind gängige RAW-Formate, Bilder, Videos und PDFs je nach Ordner.
  4. Das System erstellt automatisch die korrekte Ordnerstruktur auf dem Server und prüft auf Duplikate.
  5. Sollten Sie Dateien nachliefern, wählen Sie im Modus 'Nachlieferung in bestehenden Ordner'.
Dashboard & Auftragsverwaltung

Als Administrator haben Sie Zugriff auf alle Aufträge über admin.html.

  • Details/Bearbeiten: Termine, Fotografen oder Preise manuell anpassen.
  • Drucken: Erstellen Sie eine druckfreundliche PDF. Diese enthält einen QR-Code für den schnellen digitalen Zugriff.
  • Bestätigungs-Email: Bei Bedarf kann die Kunden-Bestätigung per Klick erneut gesendet werden.
  • Upload: Sie können Material hochladen, Ordner leeren oder Unterordner erstellen.
Manuelle Buchungen & Kundenverwaltung

Neue Bestellung: Gehen Sie in den Tab 'Kunden' und klicken Sie auf das Plus-Symbol bei einem Kunden. Die Rechnungsadresse wird automatisch übernommen.

Kundenprofil: Verwalten Sie Firmennamen, E-Mails und Adressen der Kunden zentral.

Kundenkontakte: Jedem Kunden können mehrere Ansprechpartner (Kontakte) zugeordnet werden. Öffnen Sie den Kunden und scrollen Sie zum Abschnitt 'Kontakte', um Kontakte hinzuzufügen, zu bearbeiten oder zu verknüpfen.

Als Kunde anzeigen: Mit dem Button 'Als Kunde anzeigen' in der Kundenliste öffnen Sie das Kundenportal direkt in der Ansicht dieses Kunden – nützlich zur Prüfung oder Unterstützung.

Mitarbeiter & Aktivitätsprotokoll
  • Mitarbeiter anlegen: Definieren Sie Skills, Einzugsgebiete und Sprachen, welche für das automatische Matching wichtig sind.
  • Aktivitätsprotokoll: Rechts oben finden Sie das Log. Es zeichnet auf, wer wann welche Aktion (Uploads, Statuswechsel etc.) durchgeführt hat (inkl. CSV-Export).
Uploads & NAS Struktur
  • Hauptpfad: /volume1/PROPUS DRIVE/UPLOAD/BOOKING_UPLOAD.
  • Das System prüft auf Duplikate beim Hochladen.
  • Sie können als Admin ganze Ordner (z.B. 'Zur Auswahl') leeren, bevor Sie neues Material bereitstellen.
  • Externe Kunden-Uploads über die Webseite landen in TOOL_UPLOAD.
Status 'Pausiert'

Nutzen Sie 'Pausiert' (Orange) wenn ein Termin verschoben werden muss (Wetter, Baustelle). Die Aufträge bleiben so in der Liste präsent, blockieren aber keine Kapazitäten falsch.
