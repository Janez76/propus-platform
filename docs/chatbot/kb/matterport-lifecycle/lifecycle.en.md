# Matterport Lifecycle

A Matterport tour lives on a hosted cloud server. As long as hosting is
active, the tour is reachable via its link. When hosting expires, the tour
goes offline. We offer a clear lifecycle so cost and availability stay under
your control.

## Standard runtime

A newly created tour is paid for one marketing period by default. Before
the period ends you receive a reminder with the options.

## Renewal — CHF 59

If you want the tour to stay online: hosting runs for another period
unchanged. Contents stay the same, the link stays the same, embedded
iframes continue to work without changes.

Booking: reply to the reminder e-mail or use the customer portal.

## Reactivation — CHF 74

If a tour has already been archived and should go back online: it is pulled
from the archive and made reachable again immediately — link and content
unchanged.

Reactivations make sense when the same property is being marketed again
(second tenant, new sale attempt). Re-scanning would only be needed if the
property has changed structurally.

## Archival

If the tour doesn't need to stay online: we remove it from hosting and
move it to the archive. Contents are preserved and can be reactivated later
(see above).

Archival itself is free. An archived tour is not reachable via its previous
link — visitors see an info page.

## Overview

| Action | Price | Effect |
|---|---|---|
| New tour | per booking | tour is newly created and activated |
| Renewal | CHF 59 | existing tour stays online for another period |
| Reactivation | CHF 74 | archived tour comes back online |
| Archival | free | active tour is taken offline, contents preserved |

## Common follow-ups

- **Does the tour renew automatically?** No. Renewals only happen on active
  confirmation — we don't bill without an order.
- **How long does an archived tour stay reactivatable?** Practically
  unlimited, as long as Matterport itself holds the data. In practice
  we've successfully reactivated tours years after archival.
- **Renewal after structural changes:** if the property has been remodelled,
  the old tour is technically out of date. In that case a re-scan is
  better than a renewal.
